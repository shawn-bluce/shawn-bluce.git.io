    看火影

    从小学就接触了吧，那个时候好多小伙伴用一只手抓着另一只手腕冲过去大喊“千鸟！！”的中二时光还是很棒的哈。高中毕业的那个暑假才把火影正经看起来的，一直看到了鸣人和雏田走结婚生娃，祝他们幸福。小樱也终于和佐助在一起了，也祝福他们。想吐槽的一大点，鸣人他们是商量好了一起生的孩子嘛，为啥他们的孩子都是同一级的啊。🤣

    转魔方

    最早学魔方就是因为酷炫，但是练习到酷炫的程度可远没有那么简单。以至于三阶魔方还只是停留在40s的新手水平上。不过其他的倒是了解了一点儿，二三四五阶魔方都能玩。除了常规的正阶魔方以外还有一种镜面魔方，这个是耍帅利器，因为六面都是一个颜色还高低不平，就看起来很帅，其实呢却很简单。还原镜面魔方的时候把每一面的不同高度对应到标准三阶魔方每一面的不同颜色，就可以使用还原三阶魔方的方式还原一个镜面魔方了。

    如果想要玩魔方的话不用一开始就要买个多贵的来玩，如果只是想单纯的了解一下圆自己小时候的一个梦的话就随便买个魔方就好（比较推荐磁力魔方），只要不是那种几块钱包邮的都还能用。当然如果要买来学习然后尝试更快速度的话就需要买最少两个魔方了：第一个是训练魔方，要容错能力差的那种；第二个是竞速魔方，要非常顺滑，容错能力强的那种。这样平时使用容错较差的魔方训练，能矫正自己的手法，速柠的时候就用竞速魔方可以提升速度。

    玩电脑

    接触电脑的年龄算是比较小的了吧，第一次应该是小学二年级的微机课。后来家里买了电脑就各种折腾。经常把电脑弄坏，然后找人来家里修。在修电脑的时候我就一直问问题，问东问西的，不过幸运的是每次修电脑的哥哥们都不会烦，而是认真回答我的问题。大概从那个时候我就对了解电脑的那群人有了比较好的印象。小学的时候就是玩游戏过来的，GBA 模拟器、街机模拟器、红警2、CS1.6。。。。。。记得当时家里的电脑应该是赛扬的 CPU ，配上 256m 内存和 10G 的硬盘，连 GTA 罪恶都市都玩不动，还没有网。就常跑去黑网吧玩问道和征途（幸好是小时候玩的征途，要是长大了玩怕是已经倾家荡产喽）。

    到了初中家里换了电脑还连了网，就开始玩网游、继续玩征途还开始接触了腾讯系的游戏：穿越火线和地下城。那段时间玩游戏很多，家里各种阻止我玩电脑，无所不用其极啊。给电脑设密码，藏鼠标，藏键盘，藏电源线。。。正所谓上有政策下有对策。你设密码，我学会了装双系统；你藏鼠标，我学会了快捷键；你藏键盘，我找到了软键盘；你藏电源线，我找电饭锅（是不是你也这么干过 嘿嘿）。这段时间就对电脑了解的多了些，知道了电脑里的这些硬件，学会了装系统这种。

    高一的时候，在一次偶然的机会下得知了编程这种操作，就跑去高中所在县城的新华书店花了三天饭钱买了一本《C语言从入门到精通》（没错就是从入门到精通）。不过高中是寄宿制没有电脑可以用，也就一直没有实践机会，就前两个月比较热血的时候中午不午休悄悄在教室用多媒体（就是那台放在教室里只用来放PPT的电脑）写代码，到后来没人交流还不知道下一步该干啥，就没再继续了，自然而然也就忘的一干二净了。不过高中三年的《电脑报》是我了解计算机最大的助推器了。基本上三年的电脑报从来没断过，也就是这三年的报纸让我对计算机和其他的电子产品了解的比较系统透彻。

    记得当时电脑报里会介绍手机、平板、笔记本、DIY主机、相机等等这些内容，进而了解了如何通过参数来判断硬件设备的性能这种现在看起来的”必备技能“。

    真正属于自己的电脑曾经有过四台：

    1. 神舟战神 K650D（i5-4210m + 8G + 128G + 500G） 这台是高中毕业买的，当时深知大学需要一台综（You）合（Xi）性能好一点的电脑。后来也确实体验到了比别人i3低电压的机器确实快很多；
    2. 联想小新 Air PRO （i7-7200u + 8G + 256G） 这台是大三结束的时候买的，因为大四要出来实习确实不好背着战神上下班；
    3. ThinkPad T480（i7-8250u + 32G + 512G） 这台是转正半年后买的，卖掉了上一台联想。因为我们大量使用了 Docker ，且前后端一起开发所以之前的 8G 内存确实过于吃紧。电脑买的是港版的，原配 8G + 机械硬盘，我二手卖掉了内存和硬盘又自己装了两条芝奇的 16G 内存，又换上了自己买的三星 970EVO plus，总共花了大概 9000 块钱。按理说这台机器应该能陪我征战很久很久了。哦对，因为当时找不到这台机器的介绍，我还自己录制了使用体验和拆机的视频。
    4. MacBook Pro（i7-9750H + 16G + 512G）被逼无奈的我还是从 Linux 换到了 macOS，因为工作会大量用到什么微信和企业微信这种东西，再加上我平日里就只有一台电脑用，还想学习一些视频剪辑和其他的技能，就被逼无奈换到 macOS 上来了。目前体验还行，跟 Windows 和 Linux 比起来优点缺点都很明显。

    玩游戏

    我一直都是一个喜欢玩游戏的人，不过讨厌游戏里的“竞技”，所以几乎没玩过火热的吃鸡、LOL、守望先锋这种。我觉得玩游戏就是要享受游戏过程，但是竞技类游戏经常会因为我自己的水平拖累队友的游戏体验或者被队友拖累。而且由于以前一直都没有过真正个人的时间去沉浸体验游戏，所以毕业之后顺理成章的买了一台PS4。有了自己的时间，有了各式各样优秀的游戏（战神、GTAV、神秘海域、最后的生还者、声名狼藉、COD。。。。。）自然也就重新开始了小时候从来没有机会体验的事情：一杯可乐、一台PS4、一个显示器、一个手柄、拉上窗帘带上耳机过一个完美的周末。后面又买了一台 switch 来体验世界第一任天堂带给玩家的优秀游戏体验，尤其是 switch 的独特体验，可以让我上班带着趁午休的时候玩上半个小时，也可以出门带着在地铁上玩。

    ps: 战神 真的太好玩了，除了大家都夸赞的动作元素和一镜到底以外，我感悟最深刻的一点是战神的读条。因为战神是一镜到底下来的，所以不能有读条，一有读条就打破了一镜到底的统一感。这里战神用了几个点来规避读条问题：传送门，墙缝还有”电梯“。尤其是传送门，这里的传送门相当于是两个世界的连接通道，玩家每次需要传送的时候都会先走到传送门前面然后打开传送门进入传送通道，在通道里一顿奔跑最后到达目的地。因为传送通道的场景设计很简单，所以玩家靠近传送门后，系统趁着传送门从一堆砖块到成型的时间就足够读取完通道的地图了。然后把玩家丢到通道里，让玩家奔跑，跑着跑着突然前面出现传送门出口，出去就是目的地。其实玩家奔跑的时候就是在读条，你不跑就一直等着的话最后传送门也会在你面前刷新出来。当我玩游戏发现这个点的时候就觉得非常厉害，不过后来才知道这种类似的技法其实很早就存在了，只是战神这次运用的比较好而已。

    看书

    我喜欢看各种学科的科普，尤其是理科工科的科普。还喜欢看一些所谓“乱七八糟”的书，因为看这些书的时候不会有任何功利心，没有功利心地看书是真的舒服。从来不用觉得自己看得太慢了，也不用担心自己看懂没看懂，更不用因为一个点连纠结再求证地折腾好久。

    说到看书，除了纸书之外最适合的看书设备应该就是 kindle 和 iPad 了。每个设备都能存下你“一辈子都看不完”的书，而且看起来也很方便。对于程序员来说，技术类的书籍在 iPad 上看起来要好一些，因为图可以放大，而且有 Apple pencil 的话还很方便在上面写写画画，一般大家看技术类书籍都是会有写写画画的习惯的吧。再加上买一个 goodnotes 或者 notability 跟正在看的书左右分屏，一边看书一边记笔记效率还是挺高的。

    推荐几本近期看过的好书，有兴趣可以看一看《认识电影》《程序员修炼之道－－从小工到专家》《代码整洁之道》（有两本都不错）《Just for fun》《若为自由故》《通俗天文学》
