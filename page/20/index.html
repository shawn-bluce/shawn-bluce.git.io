Java 抽象类和接口 理解抽象类和接口

发表于 2015-11-21

0X00抽象类&接口简介

抽象类

  1. abstract 抽象修饰符——抽象就是为了让子类集成的，并不能直接实现一个对象
  2. 抽象类中所有抽象方法都要在子类中实现
  3. 拥有抽象方法的类必须声明为抽象类
  4. 抽象类可以有非抽象的方法

接口

  1. interface 接口修饰符——接口是为了让类实现的
  2. 变量默认是public static final并且不能改变
  3. 方法默认是public abstract并且不能改变
  4. 接口不实现方法
